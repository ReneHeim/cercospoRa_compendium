All raw data used to run the analysis pipeline are availabe [HERE](https://owncloud.gwdg.de/index.php/s/jmlaJvWjbQKFyki). A password can be obtained on request. 

This dataset contains:

- Multispectral imagery obtained from an uncrewed aerial system (UAS). Acquired over Göttingen, Germany, in July 2022.
- Multispectral Sentinel-2 imagery (https://dataspace.copernicus.eu) acquired over Göttingen, Germany, in July 2022.
- Super-resolved Sentinel-2 imagery.
- Shape files as geopackage (.gpkg) to obtain multispectral data for the regions of interest.

Copernicus Sentinel data are made available under the European Union’s open data policy.
For more information, see: https://www.copernicus.eu/en/about-copernicus/copernicus-data-policy
